O que é o DOM?

DOM - Document Object Model: Interface de programação que representa uma página HTML como uma árvore de objetos, permitindo o JavaScript acessar e manipular dinamicamente o conteúdo, estruturas e estilo da página

document.getElementById("")

Seleciona objetos com a id que estiver entre o parenteses e os hifens do seletor

document.getElementsByClassName

Seleciona um ou mais objetos que possuem aquela classe como nome. Não modifica os objetos por si só, tem que selecionar o objeto específico para manipulação como se fosse um array, utilizando chaves []

document.querySelectorAll("")

Seleciona através de uma passagem como se escreve classe no CSS, utilizando .nomedaclasse; Para selecionar especificamente a classe é necessário nomear como array, utilizando chaves []

document.querySelector("")

Seleciona como o selector all, porem sem a necessidade de passar o elemento com chaves
