Definição de Formulários

  1. Formulários são elementos que permitem o usuário enviar dados para o Servidor
  2. Usados em várias aplicações, como login, registro, pesquisas e compras online

Estrutura básica de um formulário

  1. Um formulário é criado com a tag form que envolve os elementos de entrada
  2. Atributos importantes incluem action (URL para onde os dados serão enviados) e method (tipo de envio, como GET ou POST)

Importância da validação

  1. A validação garante que os dados enviados estejam corretos e seguros
  2. Pode ser realizada tanto no lado do cliente (JavaScript) quanto no lado do servidor

Tipos de INPUTS:

  1. text - entradas de texto
  2. password - senhas
  3. email - endereço de email
  4. checkbox e radio - seleções multiplas
  5. file - upload de arquivos

Atributos Comuns de Inputs

  1. name: identifica o input ao enviar os dados
  2. placeholder: fornece uma dica sobre o que inserir
  3. required: torna o preenchimento do campo obrigatório
  4. value: define um valor padrão para o input

Rótulos e Acessibilidade

  1. a tag label associa um rotulo a um input, melhorando a acessibilidade
  2. Utilizar rótulos ajuda na usabilidade, tornando os formulários mais fáceis de entender e usar
